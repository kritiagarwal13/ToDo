# ToDo

**Description**

This is simple ToDo app that demonstrates CRUD (Create, Read, Update, Delete) operations using Firebase Realtime Database.

**Changes Made**

 1. Added App Outline
 2. Added Components
 3. Added Firebase Realtime Database Integration via SPM
 4. Added Logic
 5. Added Checkbox

**How to Test**

1. Clone the repository from Github
2. Resolve Swift Package Manager, go to Files > Package > Resolve Package Versions
3. Run the app, create and edit tasks.

**Checklist**

1. CRUD functions
2. Checkbox selection and sort
